## 依赖

uv:

```
pip install uv
```

## 开始

1. 创建虚拟环境

```
uv venv
```

2. 激活虚拟环境

<details open>
<summary>
Windows:
</summary>

```
.venv\Scripts\activate
```

</details>

<details open>
<summary>
Linux / macOS:
</summary>

```
source .venv/bin/activate
```

</details>

3. 安装依赖

```
uv sync
```

4. 运行

<details open>
<summary>
使用 uv:
</summary>

```
cd naosql
uv run shell.py
```

</details>
